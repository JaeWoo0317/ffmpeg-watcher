## ✅ 필요한 것 (먼저 설치해야 해요!)

### 1단계. Node.js 설치

> Node.js는 이 프로그램을 실행하기 위해 꼭 필요해요.

1. 👉 [https://nodejs.org](https://nodejs.org) 에 접속합니다.
2. 초록색 **LTS** 버튼을 클릭해서 설치 파일을 다운로드합니다.
3. 다운로드된 `.pkg` 파일을 더블클릭해서 설치합니다.
   중간에 나오는 선택지는 모두 **계속** 을 눌러도 됩니다.
4. 설치가 끝나면 설치가 잘 됐는지 확인합니다:
   - `⌘ + 스페이스` 누르기 → `터미널` 입력 → Enter
   - 터미널이 열리면 아래 명령어를 입력하고 Enter:
     ```bash
     node -v
     ```
   - 아래처럼 버전 숫자가 나오면 성공이에요! ✅
     ```
     v22.0.0
     ```

---

### 2단계. Homebrew 설치

> Homebrew는 Mac에서 프로그램을 쉽게 설치할 수 있게 해주는 도구예요.
> 이미 설치되어 있다면 이 단계를 건너뛰세요.

1. 터미널을 열고 아래 명령어를 **한 줄로** 입력하고 Enter:
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```
2. 설치 중간에 비밀번호를 물어보면 **Mac 로그인 비밀번호**를 입력합니다.
   (입력해도 화면에 글자가 보이지 않는 게 정상이에요!)
3. 설치가 끝나면 확인합니다:
   ```bash
   brew -v
   ```
   버전 숫자가 나오면 성공이에요! ✅

   > ⚠️ M1/M2/M3 칩 Mac이라면 설치 후 아래 메시지가 뜰 수 있어요:
   > `Run these two commands in your terminal to add Homebrew to your PATH`
   > 그 아래에 있는 두 줄의 명령어를 복사해서 실행해주세요.

---

### 3단계. FFmpeg 설치

> FFmpeg는 실제로 영상을 변환해주는 도구예요.

1. 터미널에서 아래 명령어를 입력하고 Enter:
   ```bash
   brew install ffmpeg
   ```
2. 설치에 시간이 좀 걸릴 수 있어요. 기다려주세요.
3. 설치 완료 후 확인합니다:
   ```bash
   ffmpeg -version
   ```
   버전 정보가 쭉 나오면 성공이에요! ✅

## 📥 프로그램 다운로드

### 방법 1 — ZIP으로 다운로드 (가장 쉬움)

1. 👉 [https://github.com/JaeWoo0317/ffmpeg-watcher](https://github.com/JaeWoo0317/ffmpeg-watcher) 접속
2. 초록색 **`< > Code`** 버튼 클릭
3. **Download ZIP** 클릭
4. 다운로드된 ZIP 파일을 원하는 곳에 압축 해제합니다.
   예: `/Users/내이름/Desktop/ffmpeg-watcher`

### 방법 2 — Git으로 다운로드

터미널에서:
```bash
git clone https://github.com/JaeWoo0317/ffmpeg-watcher.git
```

## ⚙️ 설치하기

> 📌 ffmpeg-videomaker 프로젝트도 함께 있어야 합니다!
> ```
> 📂 ryu_ws/
> ├── 📂 ffmpeg-watcher/   ← 이 프로그램
> └── 📂 videomaker/       ← 이것도 필요해요!
> ```
> videomaker를 먼저 다운로드하세요: [https://github.com/JaeWoo0317/ffmpeg-videomaker](https://github.com/JaeWoo0317/ffmpeg-videomaker)

### 방법 1 — setup.command 더블클릭 (쉬움)

1. `ffmpeg-watcher` 폴더를 Finder에서 엽니다.
2. **`setup.command`** 파일을 **오른쪽 클릭** → **열기** 클릭합니다.

   > ⚠️ 처음 실행할 때 "확인되지 않은 개발자" 경고가 뜨면:
   > **오른쪽 클릭** → **열기** → **열기** 버튼 클릭

3. 터미널 창이 열리면서 자동으로 필요한 파일들을 설치합니다.
4. `설치 완료! start.command를 더블클릭하세요` 메시지가 나오면 완료! ✅

### 방법 2 — 터미널 사용

```bash
cd ~/Desktop/ffmpeg-watcher
chmod +x setup.sh && ./setup.sh
```

## ▶️ 실행하기

### 방법 1 — start.command 더블클릭 (쉬움)

1. `ffmpeg-watcher` 폴더에서 **`start.command`** 파일을 **더블클릭**합니다.

   > ⚠️ 처음 실행할 때 경고가 뜨면:
   > **오른쪽 클릭** → **열기** → **열기** 버튼 클릭
   > (한 번만 하면 그 다음부터는 더블클릭으로 바로 실행돼요!)

2. 터미널 창(서버)이 열립니다.
3. 잠시 후 브라우저에서 설정 페이지가 **자동으로 열립니다**.

   > 브라우저가 자동으로 안 열리면 직접 접속하세요:
   > 👉 **http://localhost:4001**

### 방법 2 — 터미널 사용

```bash
cd ~/Desktop/ffmpeg-watcher
node server.js
```

## 🚀 사용하기

1. **감시 폴더** 경로를 설정하고 저장합니다.
2. Finder에서 감시 폴더를 열고, 변환하고 싶은 영상 파일을 **안에 넣습니다**.
3. 브라우저 하단 **실시간 로그** 창에서 변환 진행 상황을 확인합니다!

```
[10:30:01] 감시 시작: /Users/내이름/Movies/input
[10:30:45] 감지됨: my_video.mp4
[10:31:20] 완료: my_video.mp4 → my_video.mp4  ✅
```

4. 변환 완료 후:
   - 📂 **출력 폴더** → 변환된 파일
   - 📂 **감시 폴더/done/** → 원본 파일 (자동 이동)

## ❓ 자주 묻는 질문

**Q. 브라우저가 자동으로 안 열려요!**
A. 브라우저 주소창에 직접 입력하세요: `http://localhost:4001`

**Q. "command not found: node" 오류가 나요!**
A. Node.js가 제대로 설치되지 않은 것입니다. Node.js를 다시 설치하고 터미널을 **새로 열고** `node -v`를 다시 확인하세요.

**Q. "command not found: ffmpeg" 오류가 나요!**
A. FFmpeg가 설치되지 않은 것입니다. `brew install ffmpeg`를 실행하고, M1/M2/M3 Mac이라면 Homebrew PATH 설정을 확인해주세요.

**Q. "확인되지 않은 개발자" 경고가 떠요!**
A. 파일을 **오른쪽 클릭** → **열기** → **열기** 버튼을 클릭하세요. 한 번만 하면 그 다음부터는 더블클릭으로 바로 열려요.

**Q. M1/M2/M3 Mac에서 ffmpeg가 안 찾아져요!**
A. 터미널에서 아래 명령어를 실행하세요:
```bash
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zshrc
source ~/.zshrc
```
그 후 `ffmpeg -version`을 다시 확인해보세요.

**Q. 변환이 시작되지 않아요!**
A. 아래를 확인하세요:
- 감시 폴더 경로가 올바른지 확인
- `done` 폴더 안에 파일을 넣지는 않았는지 확인
- 지원하는 영상 파일인지 확인 (`.mp4`, `.mkv`, `.avi`, `.mov`, `.wmv`, `.flv`, `.webm`, `.m4v`, `.ts`)
