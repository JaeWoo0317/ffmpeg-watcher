# 📁 FFmpeg Watcher

**영상 파일을 폴더에 넣으면 자동으로 변환해주는 프로그램입니다.**

폴더를 감시하고 있다가, 새 영상 파일이 들어오면 내가 정해둔 설정으로 자동 변환해줍니다.
설정은 브라우저에서 쉽게 할 수 있어요!

---

## 🖥️ 이 프로그램이 하는 일

```
📂 감시 폴더에 영상 파일 넣기
        ↓
🤖 자동으로 변환 시작
        ↓
📂 출력 폴더에 변환된 파일 저장
        ↓
📂 원본 파일은 done 폴더로 이동
```

---

## ✅ 필요한 것 (먼저 설치해야 해요!)

### 1. Node.js 설치

> Node.js는 이 프로그램을 실행하기 위해 꼭 필요해요.

**Windows / Mac 공통**

1. 👉 [https://nodejs.org](https://nodejs.org) 에 접속합니다.
2. **LTS** 버튼을 클릭해서 설치 파일을 다운로드합니다.

   ![nodejs 사이트에서 LTS 버튼 클릭](https://nodejs.org/static/images/logo.svg)

3. 다운로드된 파일을 더블클릭해서 설치합니다.
   중간에 나오는 선택지는 모두 **Next(다음)** 을 눌러도 됩니다.

4. 설치가 끝나면 확인해봅니다:
   - **Windows**: 검색창에 `cmd` 입력 → 명령 프롬프트 열기 → `node -v` 입력
   - **Mac**: Spotlight(⌘+Space) → `터미널` 입력 → 터미널 열기 → `node -v` 입력

   아래처럼 버전 숫자가 나오면 성공이에요! ✅
   ```
   v22.0.0
   ```

### 2. FFmpeg 설치

> FFmpeg는 실제로 영상을 변환해주는 도구예요.

#### Windows에서 설치하기

**방법 1 — winget 사용 (권장, 가장 쉬움)**

1. 검색창에 `cmd` 입력 → **명령 프롬프트**를 **관리자 권한**으로 실행합니다.
   (검색 결과에서 오른쪽 클릭 → "관리자 권한으로 실행")
2. 아래 명령어를 입력하고 Enter:
   ```
   winget install ffmpeg
   ```
3. 설치가 끝나면 새 명령 프롬프트를 열고 확인:
   ```
   ffmpeg -version
   ```
   버전 정보가 나오면 성공이에요! ✅

**방법 2 — 직접 설치**

1. 👉 [https://www.gyan.dev/ffmpeg/builds/](https://www.gyan.dev/ffmpeg/builds/) 에 접속합니다.
2. **ffmpeg-release-essentials.zip** 을 클릭해서 다운로드합니다.
3. 다운로드된 zip 파일을 압축 해제합니다.
4. 압축 해제된 폴더 안에서 `bin` 폴더를 찾습니다.
5. `bin` 폴더 안에 있는 `ffmpeg.exe` 파일이 있는 경로를 **시스템 환경 변수 PATH**에 추가합니다.
   - 검색창에 `환경 변수` 입력 → "시스템 환경 변수 편집" 클릭
   - "환경 변수" 버튼 클릭
   - 아래쪽 "시스템 변수"에서 `Path` 선택 → "편집" 클릭
   - "새로 만들기" 클릭 → `bin` 폴더 경로 붙여넣기 (예: `C:\ffmpeg\bin`)
   - 확인 버튼 클릭

#### Mac에서 설치하기

1. Spotlight(⌘+Space) → `터미널` 입력 → 터미널 열기
2. 아래 명령어를 입력하고 Enter:

   **Homebrew가 없다면 먼저 설치:**
   ```bash
   /bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
   ```

   **FFmpeg 설치:**
   ```bash
   brew install ffmpeg
   ```
3. 설치 확인:
   ```bash
   ffmpeg -version
   ```
   버전 정보가 나오면 성공이에요! ✅

## ⚙️ 설치하기

### Windows

1. 다운로드한 폴더를 엽니다.
2. **`setup.bat`** 파일을 **더블클릭**합니다.
3. 검은 창이 열리면서 자동으로 필요한 파일들을 설치합니다.
4. `완료!` 메시지가 나오면 설치 끝이에요! ✅

### Mac

1. 터미널을 열고 ffmpeg-watcher 폴더로 이동합니다:
   ```bash
   cd ~/Desktop/ffmpeg-watcher
   ```
2. 아래 명령어를 입력합니다:
   ```bash
   chmod +x setup.sh && ./setup.sh
   ```
3. 완료 메시지가 나오면 설치 끝이에요! ✅
   이후부터는 **`setup.command`** 파일을 더블클릭해서 설치할 수 있어요.

## 🎛️ 설정하기

브라우저에서 설정 페이지가 열리면 아래처럼 생겼어요:

```
┌─────────────────────────────────────┐
│  🟢 FFmpeg Watcher   [GPU 없음]     │
├────┬────┬────┬──────────┬──────────┤
│경로│영상│오디오│트림/크롭│자막/워터마크│
├─────────────────────────────────────┤
│  감시 폴더: [C:/Videos/input      ] │
│  출력 폴더: [C:/Videos/output     ] │
│                                     │
│          [ 설정 저장 ]              │
└─────────────────────────────────────┘
```

### 📂 경로 탭 (가장 중요!)

| 항목 | 설명 |
|------|------|
| **감시 폴더** | 영상 파일을 넣을 폴더 경로 입력 |
| **출력 폴더** | 변환된 파일이 저장될 폴더 경로 입력 |

> ⚠️ 경로를 입력할 때 폴더가 없으면 자동으로 만들어져요!

**경로 입력 예시:**
- Windows: `C:/Users/내이름/Videos/input`
- Mac: `/Users/내이름/Movies/input`

---

### 🎬 영상 탭

| 항목 | 설명 | 기본값 |
|------|------|--------|
| **코덱** | 영상을 어떤 방식으로 압축할지 | H.264 |
| **컨테이너** | 파일 형식 (확장자) | MP4 |
| **프로파일** | H.264/H.265 사용 시 품질 프로파일 | High |
| **해상도** | 출력 영상 크기 | 원본 유지 |
| **FPS** | 초당 프레임 수 | 원본 유지 |
| **비트레이트 모드** | CRF(품질 우선) / CBR(크기 우선) | CRF |
| **CRF 값** | 숫자가 낮을수록 고화질, 높을수록 작은 파일 (0~51) | 23 |
| **인코딩 속도** | 느릴수록 파일이 작아지지만 시간이 오래 걸림 | medium |
| **GPU 가속** | 그래픽카드를 활용해 빠르게 변환 | 자동 |

**코덱 선택 가이드:**
- 📱 **H.264** → 가장 호환성 높음, 어디서든 재생 가능 (추천!)
- 💾 **H.265** → H.264보다 파일 크기 절반, 최신 기기에서 재생
- 🌐 **AV1** → 가장 작은 파일, 변환 속도 느림
- 📹 **Copy** → 재인코딩 없이 컨테이너만 변경 (가장 빠름)

## 🚀 사용하기

1. 설정에서 **감시 폴더**와 **출력 폴더** 경로를 입력하고 저장합니다.
2. 탐색기(Finder)에서 **감시 폴더**를 엽니다.
3. 변환하고 싶은 영상 파일을 감시 폴더 안에 **복사하거나 이동**합니다.
4. 브라우저 하단 **실시간 로그** 창에서 변환 진행 상황을 확인합니다!

```
[10:30:01] 감시 시작: C:/Videos/input
[10:30:45] 감지됨: my_video.mp4
[10:30:47] (변환 진행 중...) 45%
[10:31:20] 완료: my_video.mp4 → my_video.mp4  ✅
```

5. 변환이 완료되면:
   - 📂 **출력 폴더** → 변환된 파일
   - 📂 **감시 폴더/done/** → 원본 파일

## ❓ 자주 묻는 질문

**Q. 브라우저가 자동으로 안 열려요!**
A. 브라우저 주소창에 직접 입력하세요: `http://localhost:4001`

**Q. "node를 찾을 수 없습니다" 오류가 나요!**
A. Node.js가 설치되지 않았거나 설치 후 컴퓨터를 재시작하지 않은 것입니다.
Node.js를 설치한 후 컴퓨터를 **재시작**하고 다시 시도하세요.

**Q. "ffmpeg를 찾을 수 없습니다" 오류가 나요!**
A. FFmpeg가 설치되지 않은 것입니다. 위의 FFmpeg 설치 방법을 따라주세요.

**Q. 변환이 시작되지 않아요!**
A. 아래를 확인해보세요:
- 감시 폴더 경로가 올바른지 확인
- 파일 형식이 지원되는지 확인 (`.mp4`, `.mkv`, `.avi`, `.mov`, `.wmv`, `.flv`, `.webm`, `.m4v`, `.ts`)
- `done` 폴더 안에 파일을 넣지는 않았는지 확인

**Q. Mac에서 "개발자를 확인할 수 없음" 경고가 나요!**
A. 파일을 **오른쪽 클릭** → **열기** → **열기** 버튼을 클릭하세요.

**Q. 설정을 바꾸고 싶어요!**
A. 서버가 실행 중인 상태에서 브라우저에서 설정을 변경하고 **설정 저장**을 누르면 즉시 적용됩니다!
